\section{Views}
This section will describe the view layer and what it does. For more detail about Model-View-Controller architecture, see \secref{sec:mvc}.

In \lstref{lst:homeIndexView} a truncated version of the home index view can be seen. Most of it is normal HTML, but the important part is how it utilizes variables set in the controller, that once the view is included it gets access to. This allows it to use them to generate more HTML as can be seen lines 9--13, where it uses the array of stations, to output option elements in the html.
